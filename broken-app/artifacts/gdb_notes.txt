# Шаг 2 — отладчик (gdb/lldb) на логических багах

## averages_only_positive

  gdb target/debug/deps/integration-<hash>
  break broken_app::average_positive
  run --test averages_only_positive

Наблюдение: sum считает все элементы (-5+5+15=15), делитель values.len()==3 → 5.0 вместо 10.0.
Ожидание: только положительные (5, 15) → sum=20, len=2 → 10.0.
